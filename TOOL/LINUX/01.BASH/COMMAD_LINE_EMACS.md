# Bashコマンドラインの操作 (emacsスタイル)

set -o emacs

キー		動作
Ctrl + c	コマンドなどの処理を強制終了    

Ctrl + z	コマンドなどの処理を一時停止  

fg		一時停止したコマンドを再度実行 

  

TAB		 ファイル名の補完

Ctrl + a	カーソルを行頭へ  

Ctrl + e	カーソルを行末へ  

Ctrl + b	カーソルを1文字戻る 

Ctrl + f	カーソルを1文字進める  

Ctrl + k	カーソル位置から行末までを削除  

Ctrl + u	カーソル位置から行頭までを削除  

Ctrl + d	カーソル位置の文字を削除  

Ctrl + w	カーソル位置から単語の先頭までを削除

Ctrl + y	直前に削除した文字列を張り付ける



!?COMMAND	途中まで入力されたコマンド名から、マッチした最後に実行されたコマンドが再実行される。

!?			直前に使用したコマンドの引数を再利用する (引数を利用するところにカーソルを持って行ってから)



Ctrl + l	画面内容をクリア  

ESC + u カーソル位置から行の末尾までの文字列をすべて大文字に

ESC + l カーソル位置から行の末尾までの文字列をすべて小文字に

ESC + F	カーソルを1単語先に進める  

ESC + B	カーソルを1単語戻す
