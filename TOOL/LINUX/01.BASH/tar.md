## 拡張子が.tarの場合

```
$ tar xvf v1.7_HiHi_AGCM_d_monit_a_20160601.tar
```



## 拡張子がtar.gzの場合

```
$ tar xzvf v1.7_HiHi_AGCM_d_monit_a_20160701.tar.gz 
```
